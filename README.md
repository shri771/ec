# ⚡ NexCart

A premium tech e-commerce app built with React + Vite.

## Features
- 🔍 Live search & category filtering
- 🛒 Cart drawer with quantity controls & checkout
- 💰 INR pricing (₹) with 12 dummy products
- 🎨 Dark UI with animations

## Run Locally

```bash
npm install
npm run dev
```

Open **http://localhost:5173**

## Stack
React · Vite · Context API · Vanilla CSS
